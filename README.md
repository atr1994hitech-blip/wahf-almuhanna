# منصة أرشيف وقف المهنا 🕌

منصة ويب احترافية لإدارة وأرشفة الملفات والوثائق للأوقاف.

## 🎯 الميزات الرئيسية

✅ واجهة عربية 100% (RTL)  
✅ تسجيل دخول آمن بالبريد الإلكتروني  
✅ إنشاء وإدارة أقسام ديناميكية  
✅ رفع وتنظيم الملفات والمرفقات  
✅ بحث سريع وذكي  
✅ عرض منبثق للملفات مع خيارات تعديل وحذف  
✅ لوحة إدارة للمدراء  
✅ تصميم حديث وهادئ  

## 🚀 البدء السريع

### المتطلبات
- Node.js 14+ و npm

### التثبيت والتشغيل

```bash
npm install
npm run dev
```
